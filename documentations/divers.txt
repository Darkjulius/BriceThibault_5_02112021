le fetch est une requête qui nous retourne une promesse.
Cette promesse se résout toute seule si l'appel de l'API est correct.
Il y a un .then (fonction) donc une réponse de l'appel en JSON (format textuel).
Cette méthode JSON retourne une nouvelle promesse donc une nouvelle fonction .then qui contiendra les données.

Une API est une interface avec laquelle, je vais aller chercher des données.

La lecture du code JS se lit de haut en bas.
Si il y a une erreur dans le code à n'importe quelle ligne. Il y a une erreur.

Une promesse est une valeur qui peut être disponible maintenant, dans le futur ou voire jamais. Elle sera
disponible si celle-ci est résolue.
